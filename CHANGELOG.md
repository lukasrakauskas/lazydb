# Changelog

## 0.2.0 (unreleased)

- MySQL, PostgreSQL, SQLite backends
- Multi-pane TUI with connections, editor, results, schema
- SQL editor with syntax highlighting and autocomplete
- Results grid with keyboard + mouse navigation, filter, sort
- Cell editing, row insert/delete
- Export CSV/JSON/JSONL
- Query cancellation (MySQL/PG)
- Config via ~/.config/lazydb/connections.toml
- Env-var references in connection fields
- SSL/TLS support (feature-gated)
- OS keychain integration (feature-gated)
- SSH tunneling via system ssh
- Per-connection query timeout
- MSSQL backend (feature-gated)
- Windows release builds
